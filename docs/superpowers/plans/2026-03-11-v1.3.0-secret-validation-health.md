# v1.3.0 — Secret Validation & Health Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `vaultic template sync`, `vaultic validate` (with regex pattern support), and secret age tracking to Vaultic v1.3.0.

**Architecture:** Three independent features sharing the same config section. `template sync` generates `.env.template` from decrypted environments (union of all keys, values stripped). `validate` reads the local `.env` and checks each key against rules defined in `[validation]` in `config.toml`. Secret age tracking queries the existing audit log to derive when each secret was last modified and warns if rotation policy is exceeded.

**Tech Stack:** Rust 2024 edition, clap 4 (subcommands), serde/toml (config), regex crate (new dep for pattern validation), chrono (age tracking from audit log), existing hexagonal architecture (core/traits/adapters/cli layers).

**Design decisions confirmed:**
- `template sync`: union of keys from ALL environments (A)
- `validate`: validates local `.env` file on disk, no private key required (A)
- `pattern` + `type`: combinable — all conditions validated, all failures reported (A)

use std::path::PathBuf;

use crate::adapters::cipher::age_backend::AgeBackend;
use crate::adapters::cipher::gpg_backend::GpgBackend;
use crate::adapters::key_stores::file_key_store::FileKeyStore;
use crate::adapters::parsers::dotenv_parser::DotenvParser;
use crate::cli::output;
use crate::config::app_config::AppConfig;
use crate::core::errors::{Result, VaulticError};
use crate::core::models::secret_file::SecretFile;
use crate::core::services::encryption_service::EncryptionService;
use crate::core::services::template_sync_service::TemplateSyncService;
use crate::core::traits::cipher::CipherBackend;
use crate::core::traits::parser::ConfigParser;

/// Execute `vaultic template sync`.
///
/// Decrypts all environments in memory (union of keys, values stripped),
/// and writes the result as `.env.template` (or `output_path` if given).
pub fn execute(env: Option<&str>, cipher: &str, output_path: Option<&str>) -> Result<()> {
    let vaultic_dir = crate::cli::context::vaultic_dir();
    if !vaultic_dir.exists() {
        return Err(VaulticError::InvalidConfig {
            detail: "Vaultic not initialized. Run 'vaultic init' first.".into(),
        });
    }

    let config = AppConfig::load(&vaultic_dir)?;
    let key_store = FileKeyStore::new(vaultic_dir.join("recipients.txt"));

    // Collect env names: specific env or all envs
    let env_names: Vec<String> = if let Some(e) = env {
        vec![e.to_string()]
    } else {
        config.environments.keys().cloned().collect()
    };

    if env_names.is_empty() {
        return Err(VaulticError::InvalidConfig {
            detail: "No environments defined in config.toml. Add at least one [environments] entry.".into(),
        });
    }

    let sp = output::spinner("Decrypting environments for template sync...");

    let mut secret_files: Vec<SecretFile> = Vec::new();
    let parser = DotenvParser;

    for env_name in &env_names {
        let enc_path = vaultic_dir.join(format!("{env_name}.env.enc"));
        if !enc_path.exists() {
            output::detail(&format!("Skipping {env_name}: no encrypted file found"));
            continue;
        }

        let plaintext = match cipher {
            "age" => {
                let identity_path = AgeBackend::default_identity_path()?;
                if !identity_path.exists() {
                    return Err(VaulticError::EncryptionFailed {
                        reason: format!(
                            "No private key found at {}\n\n  Run 'vaultic keys setup' to generate a key.",
                            identity_path.display()
                        ),
                    });
                }
                let backend = AgeBackend::new(identity_path);
                let svc = EncryptionService { cipher: backend, key_store: key_store.clone() };
                svc.decrypt_to_bytes(&enc_path)?
            }
            "gpg" => {
                let backend = GpgBackend::new();
                let svc = EncryptionService { cipher: backend, key_store: key_store.clone() };
                svc.decrypt_to_bytes(&enc_path)?
            }
            other => return Err(VaulticError::InvalidConfig {
                detail: format!("Unknown cipher: '{other}'. Use 'age' or 'gpg'."),
            }),
        };

        let content = String::from_utf8_lossy(&plaintext).to_string();
        let sf = parser.parse(&content)?;
        secret_files.push(sf);
    }

    output::finish_spinner(sp, "Environments decrypted");

    if secret_files.is_empty() {
        return Err(VaulticError::InvalidConfig {
            detail: "No encrypted environments found to sync from.\n\n  \
                     Run 'vaultic encrypt' first to create encrypted files.".into(),
        });
    }

    let template = TemplateSyncService::merge_to_template(&secret_files);
    let template_content = parser.serialize(&template)?;

    let dest = output_path
        .map(PathBuf::from)
        .unwrap_or_else(|| PathBuf::from(".env.template"));

    std::fs::write(&dest, template_content)?;

    let key_count = template.keys().count();
    output::success(&format!(
        "Written {} keys to {}",
        key_count,
        dest.display()
    ));
    println!("\n  All values stripped — safe to commit.");
    println!("  Run 'vaultic check' to verify your local .env is in sync.");

    super::audit_helpers::log_audit(
        crate::core::models::audit_entry::AuditAction::TemplateSync,
        vec![dest.to_string_lossy().to_string()],
        Some(format!("{key_count} keys from {} environments", env_names.len())),
    );

    Ok(())
}
```

        for (key, rule) in rules {
            let mut failures: Vec<String> = Vec::new();
            let value = values.get(key);

            // required check
            if rule.required && (value.is_none() || value.map(|v| v.is_empty()).unwrap_or(true)) {
                failures.push(format!("required but missing or empty"));
                results.push(KeyValidation { key: key.clone(), passed: false, failures });
                continue; // no further checks if absent
            }

            // If not required and absent, skip
            let Some(val) = value else {
                continue;
            };

            // type check
            if let Some(ref t) = rule.value_type {
                match t.as_str() {
                    "url" => {
                        if !val.starts_with("http://") && !val.starts_with("https://") && !val.contains("://") {
                            failures.push("expected a valid URL (must contain '://')".to_string());
                        }
                    }
                    "integer" => {
                        match val.parse::<i64>() {
                            Err(_) => failures.push(format!("expected integer, got '{val}'")),
                            Ok(n) => {
                                if let Some(min) = rule.min {
                                    if n < min { failures.push(format!("value {n} below minimum {min}")); }
                                }
                                if let Some(max) = rule.max {
                                    if n > max { failures.push(format!("value {n} above maximum {max}")); }
                                }
                            }
                        }
                    }
                    "boolean" => {
                        let lower = val.to_lowercase();
                        if !matches!(lower.as_str(), "true" | "false" | "1" | "0" | "yes" | "no") {
                            failures.push(format!("expected boolean (true/false/1/0/yes/no), got '{val}'"));
                        }
                    }
                    "string" | _ => {
                        // min_length / max_length handled below
                    }
                }
            }

            // min_length
            if let Some(min_len) = rule.min_length {
                if val.len() < min_len {
                    failures.push(format!("too short ({} chars, minimum {min_len})", val.len()));
                }
            }

            // max_length
            if let Some(max_len) = rule.max_length {
                if val.len() > max_len {
                    failures.push(format!("too long ({} chars, maximum {max_len})", val.len()));
                }
            }

            // pattern
            if let Some(ref pattern) = rule.pattern {
                let re = Regex::new(pattern).map_err(|e| VaulticError::InvalidPattern {
                    key: key.clone(),
                    pattern: pattern.clone(),
                    reason: e.to_string(),
                })?;
                if !re.is_match(val) {
                    failures.push(format!("does not match pattern \"{pattern}\""));
                }
            }

            let passed = failures.is_empty();
            results.push(KeyValidation { key: key.clone(), passed, failures });
        }

        Ok(ValidationReport { results })
    }
}

use std::collections::HashMap;
use std::path::Path;

use crate::adapters::parsers::dotenv_parser::DotenvParser;
use crate::cli::output;
use crate::config::app_config::AppConfig;
use crate::core::errors::{Result, VaulticError};
use crate::core::services::validation_service::ValidationService;
use crate::core::traits::parser::ConfigParser;

/// Execute `vaultic validate`.
///
/// Reads the local `.env` (or `file` if given) and validates each key
/// against rules defined in `[validation]` in `.vaultic/config.toml`.
pub fn execute(file: Option<&str>) -> Result<()> {
    let env_path = Path::new(file.unwrap_or(".env"));

    if !env_path.exists() {
        return Err(VaulticError::FileNotFound {
            path: env_path.to_path_buf(),
        });
    }

    let vaultic_dir = crate::cli::context::vaultic_dir();
    if !vaultic_dir.exists() {
        return Err(VaulticError::InvalidConfig {
            detail: "Vaultic not initialized. Run 'vaultic init' first.".into(),
        });
    }

    let config = AppConfig::load(&vaultic_dir)?;

    let rules = match config.validation {
        Some(ref v) if !v.is_empty() => v,
        _ => {
            output::warning("No [validation] rules found in .vaultic/config.toml");
            println!("\n  Add rules under [validation] to validate your secrets:");
            println!("  [validation]");
            println!("  DATABASE_URL = {{ type = \"url\", required = true }}");
            println!("  PORT = {{ type = \"integer\", min = 1024, max = 65535 }}");
            return Ok(());
        }
    };

    let parser = DotenvParser;
    let content = std::fs::read_to_string(env_path)?;
    let secret_file = parser.parse(&content)?;

    // Build flat map key→value for ValidationService
    let values: HashMap<String, String> = secret_file
        .keys()
        .map(|k| (k.clone(), secret_file.get(k).cloned().unwrap_or_default()))
        .collect();

    output::header("🔍 vaultic validate");
    output::detail(&format!("File: {}", env_path.display()));
    output::detail(&format!("Rules: {} defined", rules.len()));

    let report = ValidationService::validate(&values, rules)?;

    // Sort results: failures first, then passes, both alphabetically
    let mut results = report.results.clone();
    results.sort_by(|a, b| {
        match (a.passed, b.passed) {
            (false, true) => std::cmp::Ordering::Less,
            (true, false) => std::cmp::Ordering::Greater,
            _ => a.key.cmp(&b.key),
        }
    });

    for r in &results {
        if r.passed {
            output::success(&format!("{} — ok", r.key));
        } else {
            let reasons = r.failures.join("; ");
            output::error_line(&format!("{} — {}", r.key, reasons));
        }
    }

    println!();

    let failed = report.failed_count();
    let total = report.results.len();
    let passed = total - failed;

    if report.is_ok() {
        output::success(&format!("{passed}/{total} rules passed — all good"));
    } else {
        output::warning(&format!("{passed}/{total} rules passed, {failed} failed"));
        println!("\n  Fix the values in your .env and run 'vaultic validate' again.");

        // Audit + exit with non-zero for CI
        super::audit_helpers::log_audit(
            crate::core::models::audit_entry::AuditAction::Validate,
            vec![env_path.to_string_lossy().to_string()],
            Some(format!("{passed}/{total} passed, {failed} failed")),
        );

        return Err(VaulticError::ValidationFailed { count: failed });
    }

    super::audit_helpers::log_audit(
        crate::core::models::audit_entry::AuditAction::Validate,
        vec![env_path.to_string_lossy().to_string()],
        Some(format!("{passed}/{total} passed")),
    );

    Ok(())
}
```
