## Modelo de Licencia Dual

Vaultic está disponible bajo dos opciones de licencia:

### 1. Open Source — AGPLv3

La [GNU Affero General Public License v3.0](../LICENSE) te permite usar, modificar y distribuir Vaultic libremente, siempre que:

- Cualquier modificación que realices se publique también bajo AGPLv3
- Si ejecutas una versión modificada como servicio de red, debes proporcionar el código fuente a tus usuarios
- Todas las obras derivadas permanezcan open source bajo los mismos términos

**La AGPLv3 es suficiente si:**

- Usas Vaultic internamente en tu equipo tal cual (sin modificaciones)
- Contribuyes tus modificaciones de vuelta al proyecto open source
- Construyes herramientas open source que se integren con Vaultic

### 2. Licencia Comercial

La licencia comercial elimina las obligaciones de copyleft de AGPLv3.

**Necesitas una licencia comercial si:**

- Integras Vaultic en software o servicios propietarios
- Modificas Vaultic y no puedes liberar las modificaciones como open source
- Ofreces Vaultic como parte de un SaaS, PaaS o servicio gestionado comercial
- Requieres términos específicos de soporte, SLAs o indemnización

## Qué Incluye la Licencia Comercial

- Permiso para usar, modificar y distribuir Vaultic sin obligaciones AGPLv3
- Libertad para mantener modificaciones e integraciones como propietarias
- Soporte prioritario y comunicación directa con el equipo de desarrollo
- Términos personalizados adaptados a las necesidades de tu organización

## Preguntas Frecuentes

**¿Puedo usar Vaultic gratis en mi empresa?**
Sí. Usar la versión open source como herramienta CLI para gestionar los secretos de tu equipo está completamente permitido bajo AGPLv3. No se necesita licencia comercial para este caso de uso.

**¿Necesito licencia comercial solo por usar Vaultic internamente?**
No. Si usas Vaultic tal cual (sin modificaciones) o contribuyes cualquier modificación de vuelta al proyecto, la AGPLv3 cubre tu uso.

**¿Qué pasa si construyo un producto que usa Vaultic?**
Si tu producto incluye o distribuye Vaultic (o una versión modificada), y no puedes liberarlo bajo AGPLv3, entonces necesitas una licencia comercial.

**¿Puedo evaluar Vaultic antes de comprar?**
Por supuesto. La versión open source es completamente funcional. Evalúala bajo AGPLv3 y contáctanos cuando necesites términos comerciales.
